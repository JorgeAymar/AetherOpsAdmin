AetherOps - Intelligent VPS Administration Service

OBJECTIVE: Create an elegant, high-performance, and functional website for "AetherOps," a new generation AIOps service that provides autonomous monitoring, predictive correction, and advanced security hardening for Virtual Private Servers (VPS) and cloud infrastructure fleets.

DESIGN AND AESTHETICS

Color Palette:

Primary Background: Matte Black (#1C1C1C) - For a sophisticated, high-tech foundation.

Primary Accent: Soft Gold (#D4AF37) - For CTAs, logos, and focal highlights.

Secondary Background/Text: Cream White (#FFFEF7) - For high contrast and elegance.

Secondary Accent (Tech/Depth): Deep Teal (#004D40) - For status indicators and section highlights.

Typography: Elegant Serif for all major titles (e.g., Playfair Display style) and a clean, legible Sans-Serif for body text and technical details.

Imagery: Use abstract AI graphics, polished dashboard mockups, sophisticated network diagrams, and high-quality placeholder images of data centers/technology (use Unsplash placeholders).

Interaction: Smooth, subtle fade-in and slide-up animations on scroll for content sections. Generous padding and spacing to ensure a clean, premium, and breathable layout.
